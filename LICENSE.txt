================================================================================
                          NATURE OF PROJECT
================================================================================

This software application ("NursePath" or the "Application") is a student 
academic project developed as a learning tool for nursing students. The 
Application is designed to reinforce concepts taught in nursing coursework 
and serve as a personal study aid.

This is NOT a clinical tool. The Application mirrors textbook content and 
is intended for educational reinforcement only.
